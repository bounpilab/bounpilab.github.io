# Literary Machine Translation to Produce Translations that Reflect Translators' Style and Generate Retranslations

**Funding Agency:** Tübitak 1001 Research Project 121K221

**Dates:** 2021-2024

**Fund:** 504,270 TL

This project focused on developing machine translation systems capable of capturing and reflecting individual translators' styles, as well as generating high-quality literary retranslations that honor the artistic nature of literary works.
