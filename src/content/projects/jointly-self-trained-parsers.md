# Jointly Self-trained Parsers

**Funding Agency:** Bogazici University Research Fund, BAP

**Principal Investigator:** Arzucan Özgür

**Dates:** 2012-2013

This project focused on developing jointly self-trained parsers, exploring techniques for improving parser performance through self-training approaches that leverage multiple parsing paradigms.
