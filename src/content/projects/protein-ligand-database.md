# Protein-Ligand Database Construction

**Funding Agency:** Bogazici University Research Fund, BAP

**Principal Investigator:** Arzucan Özgür

**Dates:** 2016-2018

This project focused on constructing a comprehensive protein-ligand database by applying text mining techniques to extract protein-ligand interaction information from biomedical literature.
