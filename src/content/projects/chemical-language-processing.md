# Chemical Language Processing for Target-Based Drug Design

**Funding Agency:** The Scientific and Technological Research Council of Turkey (TUBITAK 1001)

**Principal Investigator:** Arzucan Özgür

**Dates:** 2020-2023

This project applied natural language processing techniques to chemical structures for target-based drug design. By treating molecules as "chemical language," the project developed novel approaches for predicting drug-target interactions and designing new drug candidates.
