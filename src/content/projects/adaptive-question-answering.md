# Developing an Adaptive Question Answering System Enabling Primary and Secondary Education Students Accessing Accurate and Reliable Information

**Funding Agency:** TÜBİTAK 1001 Program (FATIH Project)

**Project Leader:** Tunga Güngör

**Dates:** 2013-2016

This project developed an adaptive question answering system designed to help primary and secondary education students access accurate and reliable information through natural language questions.
