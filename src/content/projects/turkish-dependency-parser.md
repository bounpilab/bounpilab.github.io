# A Deep Learning based Turkish Dependency Parser

**Funding Agency:** TUBITAK 1005

**Dates:** 2018-2019

This project developed a state-of-the-art deep learning-based dependency parser specifically designed for Turkish. The parser addresses the unique challenges posed by Turkish's agglutinative morphology and relatively free word order.
